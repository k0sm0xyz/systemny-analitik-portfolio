# Кейс №1: Автоматизация кофейни

### Функциональные требования:
* **FR-01:** Отображение актуального меню и цен.
* **FR-02:** Возможность онлайн-предзаказа к определенному времени.
* **FR-03:** PUSH-уведомление клиенту о готовности заказа на стойке.

### User Stories:
* **US-01:** Я как клиент хочу заказать кофе по пути, чтобы не стоять в очереди.
