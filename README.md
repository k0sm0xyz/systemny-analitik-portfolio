## 🗺 Карта проектов (Таймлайн 2026)

### 📈 1. Бизнес-анализ и консалтинг

* **2026-01** 🏭 **[Кейс «ТехноПром»](./2026-01_business-analysis)**
  * *Суть:* Предпроектное обследование производственной компании (400 человек) с целью устранения 25.6% просрочек заказов.
  * *Артефакты:* План обследования, методология триангуляции данных, скрипты интервью, хронометраж Shadowing, требования To-Be.

### 📐 2. Системный анализ и проектирование систем

* **2026-02** 🐾 **[Онлайн-запись «Лапка»](./2026-02_system-analysis)**
  * *Суть:* Проектирование модуля онлайн-бронирования слотов к ветеринарам.
  * *Артефакты:* Границы системы (Vision & Scope), функциональные/нефункциональные требования, Use Cases, негативные тест-кейсы.

* **2026-02** 🏢 **[Бронирование Коворкинга](./04-coworking-case)**
  * *Суть:* Разработка системы шеринга рабочих мест в ИТ-хабе.
  * *Артефакты:* Статусная модель жизненного цикла брони, ER-диаграмма сущностей, обработка конфликтов API (`409 Conflict`).

* **2026-03** ☕ **[Умная Кофейня](./03-coffee-shop-case)**
  * *Суть:* Проектирование системы предзаказа напитков с рекомендательным ИИ-модулем и трансграничной оплатой.
  * *Артефакты:* Сквозной JSON API-контракт, UML Sequence диаграмма обработки сетевых таймаутов (`Gateway Timeout`) в платежном шлюзе **BRICS Pay**.

### 🔌 3. Тестирование, интеграции и работа со средой

* **2026-03** 🌐 **[Интеграции & API](./05-delivery-case)**
  * *Суть:* Ручное тестирование REST API интеграций и спецификации службы доставки.
  * *Артефакты:* Документация API, Postman-коллекции, валидация JSON-схем ответов.

* **2026-04** 🛴 **[IoT-шеринг самокатов «Вжух»](./08-scooter-vzhuh-case)**
  * *Суть:* Высоконагруженная платформа краткосрочной аренды самокатов с динамическим ИИ-тарифом.
  * *Артефакты:* JSON-контракт телеметрии по протоколу **MQTT**, архитектурная Mermaid-схема принятия решений ИИ-агентом, токенизация карт по стандарту PCI DSS.

* **2026-05** 🔒 **[Автоматизация обхода DPI](./09-dpi-obhod-case)**
  * *Суть:* Исследование глубокого анализа пакетов и создание локального SOCKS5-прокси на Python.
  * *Артефакты:* Спецификация комбо-стратегии фрагментации TLS-пакетов (`tpws`), JSON-контракт конфигурации обхода.
