# 📊 Терминал Системного Аналитика | Портфолио проектов

Добро пожаловать в репозиторий практических кейсов и артефактов системного и бизнес-анализа. Проекты упорядочены по хронологии реализации и охватывают ключевые домены: от предпроектного обследования заводов до проектирования сложных автоматизированных систем (HR-Tech, E-commerce, API).

## 🛠️ Профессиональный стек и навыки (Hard Skills)

* **Моделирование процессов**: BPMN 2.0, UML (Use Case, Sequence, Activity Diagram), ER-диаграммы (Crow's Foot).
* **Проектирование API**: REST API, спецификации OpenAPI/Swagger, коллекции Postman, JSON/XML контракты.
* **Базы данных & SQL**: Проектирование реляционных БД, СУБД PostgreSQL, DBeaver, написание сложных запросов (JOIN, GROUP BY, HAVING).
* **Управление требованиями**: Сбор и выявление требований (интервью, Shadowing), спецификации SRS, User Stories (Gherkin), Use Cases.
* **Инструменты**: Jira, Confluence, Draw.io, Miro, Git, GitHub Actions.
