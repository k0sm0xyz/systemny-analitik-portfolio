# Кейс №2: IoT-шеринг самокатов «Вжух»

### Функциональные требования:
* **FR-01:** Отображение самокатов на интерактивной карте.
* **FR-02:** Разблокировка замка по QR-коду через камеру.
* **FR-03:** Подсветка бесплатных парковочных зон.
* **FR-04:** Применение «тарифа для ленивых» вне зон парковки.

### Нефункциональные требования:
* **NFR-PERF:** Поддержка 3x пиковой нагрузки в часы пик и выходные.
* **NFR-LATENCY:** Время отклика интерфейса при транзакциях не более 2 секунд.
* **NFR-NET:** Обновление GPS-телеметрии 1 раз в секунду по протоколу MQTT.
* **NFR-SEC:** Полный запрет на хранение номеров карт (токенизация по PCI DSS).
