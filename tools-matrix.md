# Матрица «Задача → Инструмент»

## Сбор и анализ требований

| Задача | Инструмент | Как использую |
|--------|------------|---------------|
| Интервью с заказчиком | Telegram / MAX / Perplexity | Запись, расшифровка, подготовка вопросов |
| Фиксация требований | Markdown (GitHub) | User Stories, Acceptance Criteria |
| Визуализация процессов | Mermaid (GitHub) | BPMN, Sequence, ER-диаграммы |
| Хранение артефактов | GitHub | Портфолио, версионирование |

## Проектирование и моделирование

| Задача | Инструмент | Как использую |
|--------|------------|---------------|
| Моделирование данных | Mermaid / Draw.io | ER-диаграммы |
| Описание API | Markdown (OpenAPI-style) | API-контракты |
| Диаграммы последовательности | Mermaid | Sequence Diagram |
| Бизнес-процессы (BPMN) | Mermaid | BPMN-диаграммы |

## Управление задач

| Задача | Инструмент | Как использую |
|--------|------------|---------------|
| Трекинг задач | GitHub Issues | User Stories, баги, изменения |
| Хранение документации | GitHub (Markdown) | README, артефакты, кейсы |
| Совместная работа | GitHub Discussions / Telegram | Обсуждение, обратная связь |

## Тестирование и проверка данных

| Задача | Инструмент | Как использую |
|--------|------------|---------------|
| Тестирование API | Postman | Авторская приемка |
| Проверка данных | SQL (DBeaver) | Запросы, анализ дубликатов |
| Автотесты | GitHub Actions | Проверка бинарников, скриптов |

## ИИ-помощники

| Задача | Инструмент | Как использую |
|--------|------------|---------------|
| Подготовка к интервью | Perplexity | Поиск терминов, глоссарий |
| Генерация идей | DeepSeek | Промты, проверка артефактов |
| Поиск информации | Perplexity | Быстрое погружение в предметную область |
